# R-phoneme 🔷 [Beta]

> Инструмент для сбора данных и диагностики произношения звука «Р» —
> профессиональное приложение для детской логопедии на базе нейросети.

**[▶ Открыть в браузере — попробовать сейчас](https://lexbayart.github.io/r-phoneme/)**
Без установки. Без аккаунта. Работает офлайн.

---

## 🔷 Для чего это нужно

Приучить ребёнка произносить звук «Р» — одна из самых частых задач детского логопеда.
Но как понять, произносит ли ребёнок «Р» правильно? Как отследить прогресс?
И как улучшить саму нейросеть, которая этот звук распознаёт?

R-phoneme решает все три вопроса одновременно.

Приложение записывает речь через микрофон браузера и в реальном времени анализирует
каждый звук с помощью нейросети (Edge Impulse, скомпилированной в WASM).
Модель классифицирует входящий сигнал в один из трёх классов: **«Р»**, **шум**, **неизвестно**.

### Сбор данных для дообучения модели

Главная цель проекта — не просто «детектор», а **инструмент сбора размеченных данных**.
Каждое срабатывание фиксируется в логе с временной меткой, уверенностью модели
и пиковой амплитудой. Лог можно скопировать и проанализировать: где модель ошиблась,
где пропустила, где сработала ложно.

На основе этих логов логопед или исследователь может:
- Выявить слабые места нейросети (тихие «Р», «Р» в определённых слогах, «Р» после паузы)
- Собрать примеры для дообучения модели
- Скорректировать пороги и параметры под конкретного ребёнка

### Два режима произношения

В логопедической практике звук «Р» отрабатывается двумя способами:
- **Непрерывное** «р-р-р-р-р» — для формирования артикуляционного навыка
- **Раздельное** «р» · «р» · «р» — для закрепления в изолированном произношении

Приложение позволяет переключаться между режимами на лету — в лог попадает метка,
чтобы потом анализировать результаты по режимам отдельно.

### Почему это лучше, чем «просто послушать»

- Объективный показатель: модель выдает процент уверенности, а не субъективное «как мне кажется»
- Временные метки: видно, через сколько миллисекунд после произношения сработало обнаружение
- Настраиваемые пресеты: 5 конфигураций под разные сценарии — от агрессивного обнаружения до строгой фильтрации
- Экспорт логов: данные можно выгрузить для внешнего анализа и дообучения модели

## ✨ Возможности

- 🎤 **Захват речи в реальном времени** — через микрофон браузера, без сервера
- 🧠 **Нейросеть Edge Impulse (WASM)** — классификация «Р» / шум / неизвестно прямо в браузере
- 📊 **Визуализация уверенности** — цветные полосы для каждого класса с процентами
- 🟢 **Индикатор обнаружения** — круг с подсветкой при срабатывании
- 📈 **Уровень микрофона** — полоса громкости в реальном времени
- 🔁 **Два режима** — непрерывное «р-р-р-р-р» и раздельное «р» · «р» · «р»
- ⚙️ **5 пресетов** — готовые конфигурации под разные сценарии тестирования
- 🎚️ **Живые слайдеры** — порог, масштаб, интервал, нормализация — всё меняется на лету
- 📋 **Лог с временными метками** — каждая итерация с данными; копирование одной кнопкой
- 📋 **Копирование настроек** — для воспроизведения конфигурации
